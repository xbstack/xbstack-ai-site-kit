# Contributing

Thanks for your interest in contributing to XBSTACK AI Site Kit.

## Ways to contribute

- Fix bugs
- Improve documentation
- Add templates
- Improve SEO utilities
- Add deployment examples
- Report broken links
- Suggest content taxonomy improvements

## Development

```bash
npm install
npm run dev
npm run build
```

## Pull Request Checklist

Before submitting a PR:

* The project builds successfully
* No secrets are committed
* Documentation is updated if behavior changes
* New templates include examples
* Frontmatter follows the schema

## Code Style

Use TypeScript where possible. Keep components small and reusable.

## Maintainer Notes

Issues are triaged regularly. PRs should be focused and easy to review.
